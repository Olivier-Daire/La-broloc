# La broloc

##Requirements
###OpenGL
Version >= 3.3  
###SDL2  
```shell
sudo apt-get install libsdl2-dev
```
###Glew  
```shell
sudo appt-get install glew
```

##Installation
###1 - Clone the project
```shell
git clone git@github.com:Olivier-Daire/La-broloc.git
```

###2 - Build
```shell
mkdir La-broloc-build
cd La-broloc-build
cmake ../La-broloc
make
```

###3 - Execute  
Still inside `La-broloc-build`
```shell
./src/main
```
