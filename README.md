# openGL-Project

##Requirements
###OpenGL
Version >= 3.3  
###SDL2  
```shell
sudo apt-get install libsdl2-dev
```
###Glew  
```shell
sudo appt-get install glew
```

##Installation
###1 - Clone the project
```shell
git clone git@github.com:Olivier-Daire/openGL-Project.git
```

###2 - Build
```shell
mkdir Project-build
cd Project-build
cmake ../Projet
make
```

###3 - Execute  
Still inside `Project-build`
```shell
./src/main
```
